# Finance-Performance-Dashboard
## Overview

This project is an interactive Finance Performance Dashboard built in Power BI using the Financial Sample dataset. The dashboard provides a comprehensive view of sales performance, profitability, and business growth through KPI tracking, time intelligence calculations, trend analysis, and comparative reporting.

The primary objective of this project is to demonstrate business intelligence skills including data modeling, DAX calculations, time intelligence functions, dashboard design, and data storytelling.

## Business Problem

Organizations need a clear understanding of their financial performance across different products, countries, and customer segments. Raw financial data often makes it difficult to identify trends, monitor growth, and evaluate profitability.

This dashboard addresses these challenges by providing:

* Revenue and profit monitoring
* Month-to-Date (MTD) and Year-to-Date (YTD) performance tracking
* Year-over-Year (YoY) growth analysis
* Product and segment comparisons
* Country-wise sales performance analysis
* Interactive filtering for detailed exploration

## Dataset

Dataset: Financial Sample.xlsx

The dataset contains financial transaction data including:

| Column        | Description            |
| ------------- | ---------------------- |
| Date          | Transaction date       |
| Sales         | Total sales revenue    |
| Profit        | Profit generated       |
| Gross Sales   | Sales before discounts |
| Discounts     | Discount amount        |
| Units Sold    | Number of units sold   |
| Product       | Product name           |
| Segment       | Customer segment       |
| Country       | Country of sale        |
| Discount Band | Discount category      |
| Sale Price    | Product selling price  |
| Year          | Year of transaction    |
| Month Name    | Month name             |
| Month Number  | Month number           |

## Key Insights Generated

The dashboard helps answer business questions such as:

* What is the total revenue and profit generated?
* How is current performance compared to previous years?
* Which countries contribute the highest sales?
* Which products generate the most revenue?
* Which customer segments are most profitable?
* How do discounts affect profitability?
* Are sales improving over time?
